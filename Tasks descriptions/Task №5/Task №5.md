# Завдання №5: Обробка колекцій

Згідно цього завдання потрібно було:

 - Реалізувати можливість скасування (undo) операцій (команд).
 - Продемонструвати поняття "макрокоманда"
 - При розробці програми використовувати шаблон Singletone.
 - Забезпечити діалоговий інтерфейс із користувачем.
 - Розробити клас для тестування функціональності програми.

## Пояснення структури коду

Для реалізації завданнябуло взято код із минулої роботи, та змінено із внесенням таких змін: 

 - у Calc для реалізації шаблону Singletone використовується метод getInstance, щоб у програмі був один і той же об'єкт.
 - Додано команди undo та execute, щоб можна було відмінити або створити. Для цього добавлено два нових файла Command.java(інтерфейс із двома оголошеними методами) та AddItemCommand(файл,щоб дії macro та generate могли скасуватися після undo)
 - Створено клас MаcroCommand для створення відразу трьох записів
 - Також оновлено клас тестів, щоб тестилися також макро, анду функції та щоб після виконання не було сміття добавлено функції clear
