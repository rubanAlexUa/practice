# Завдання №2: Класи та об'єкти

Згідно цього завдання потрібно було:

- Розробити клас, що серіалізується, для зберігання параметрів і результатів обчислень Використовуючи агрегування, розробити клас для знаходження рішення задачі.
- Розробити клас для демонстрації в діалоговому режимі збереження та відновлення стану об'єкта, використовуючи серіалізацію. Показати особливості використання transient полів.
- Розробити клас для тестування коректності результатів обчислень та серіалізації/десеріалізації. Використовувати докладні коментарі для автоматичної генерації документації засобами javadoc.
- Виконати індивідуальне завдання згідно номеру в списку.

## Умова програми

_Варіант 17_

**17. Визначити 8-річне та 16-річне уявлення цілісного значення загального електричного опору трьох послідовно з'єднаних провідників при заданому постійному струмі та відомій напрузі на кожному провіднику.**

## Пояснення структури коду

Якщо коротко: то задача вирішує проблему пошукузагального опору електричного кола, що складається з рьох провідників.

Для початку. Код розподілено на три класи: _Main_, _Item_ та _Calc_. Main — показує мені та загалом збирає проект. Calc — отримує числа, рахує опір, та керує файлом. Item — просто зберігає у собі значення струму, напруги та опору, також виводе значення у 8-значному та 16-значному форматі.

Коли код запускається, то починає працювати вічний do-while цикл, що виводе меню в консоль. Щоб перейти далі, потрібно вибрати один із варіантів: s, q, v, g, r.
